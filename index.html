Spec Finder

Type a term off the drawings. It tells you which MasterFormat specification section is required — no section number needed — and which drawing sheets the term appears on, in context, with a link to the sheet. Built on the published master library (864 sections, Divisions 00–48).

Matches drawing language (including project abbreviations like HM, IGU, SCW) against the spec library, and shows the drawing sheets the term appears on with the surrounding text. Try: sliding door, grab bar, terrazzo, hm.
